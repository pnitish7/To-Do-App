# To-Do App

**This is a to-do app which has following features -** <br/>
-Add Task<br/>
-View task<br/>
-Delete Task<br/>
-Mark any task as completed<br/>
-Tasks get recovered from local storage <br/>
<br/>
<br/>
## Set up and run <br/>
**Run command`npm init`**<br/>
**Run command`npm run dev`**<br/>
